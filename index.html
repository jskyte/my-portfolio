Information System for Intangible Cultural Heritage

The intangible cultural heritage application was developed to help disseminate and preserve intangible cultural heritage in four provinces, namely DKI Jakarta, Banten, Lampung, and West Java. The application can display data about cultural heritage, aspects of cultural heritage, maestro of cultural heritage, and locations of cultural heritage. This application can also display cultural heritage locations that are closest to the user within a certain radius. This application was developed using Java, XML, Slim API, and MySQL as a database

Administrator Dashboard for PT Fun International

The administrator dashboard was developed using HTML, CSS, PHP, and MySQL as the database.
